# admetshiny 1.0.0

* Initial CRAN release.
* Interactive Shiny application with four data-source modules: SwissADME,
  CDK & webchem, ADMETlab 3.0 and Deep-PK.
* Drug-likeness filters: Lipinski, Veber, Ghose, Egan, Muegge.
* BOILED-Egg model, PCA, t-SNE, parallel coordinates, violin, radar and
  correlation heatmap visualizations.
* Tanimoto / AGNES structural-similarity clustering.
* Dark mode, custom colour palettes and high-resolution figure export.
* Codename: *Espeletia hartwegiana*.
